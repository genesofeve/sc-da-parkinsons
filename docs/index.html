Welcome


Welcome to the website designed to display the code and analysis associated with “Single-cell RNA-seq of mouse dopaminergic neurons informs candidate gene selection for sporadic Parkinson’s disease” (Hook et al., 2017)

This website contains:

  • An About section containing our paper abstract
  • A License section containing copyright information
  • An Analyses section (see navbar above) containing reproducible analysis code
  • A Paper Figures section containing reproducible figure production code
  • A Shiny app that allows for exploration of the expression data

This website was designed and created by Paul Hook.
Raw sequencing data can be found at GEO:GSE108020


bioRxiv Citation

Single-cell RNA-seq of mouse dopaminergic neurons informs candidate gene selection for sporadic Parkinson’s disease. Paul W. Hook, Sarah A. McClymont, Gabrielle H. Cannon, William D. Law, A. Jennifer Morton, Loyal A. Goff, Andrew S. McCallion. bioRxiv 148049; doi: https://doi.org/10.1101/148049
